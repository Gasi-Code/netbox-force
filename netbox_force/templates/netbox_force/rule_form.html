{{ ui.rule_type_naming|default:"Naming Convention" }}

Ensures that a field value matches a regex pattern. Uses Python's re.fullmatch(), so the entire value must match.

Examples:

  • ^[A-Z]{3}-\d{3}$ — matches ABC-123
  • ^(prod|dev|staging)-.* — must start with prefix

{{ ui.rule_type_required|default:"Required Field" }}

Ensures that a field is not empty, null, or blank. Useful for fields that are optional in NetBox by default but required by your organization.

Common models:

  • dcim.device
  • dcim.site
  • dcim.rack
  • ipam.ipaddress
  • ipam.prefix
  • tenancy.tenant
  • virtualization.virtualmachine
